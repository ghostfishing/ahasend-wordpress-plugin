# Building the Plugin

Build a zip file suitable for WordPress plugin installation or directory submission.

```bash
zip -r ahasend-email-sender.zip ahasend-email-sender.php uninstall.php readme.txt README.md LICENSE images/
```

The resulting `ahasend-email-sender.zip` can be uploaded via **Plugins > Add New > Upload Plugin** in WordPress admin, or submitted to the WordPress Plugin Directory.
